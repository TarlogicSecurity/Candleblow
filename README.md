# Candleblow
Sniffing firmware for Microchip's ATPL360-EK evaluation kit, to use in conjunction with [PLCTool](https://github.com/TarlogicSecurity/PLCTool).

## Requirements 
You will need to install the latest version version of [Atmel Studio 7](https://www.microchip.com/mplab/avr-support/avr-and-sam-downloads-archive) (Windows only) and open the solution file `Candleblow.atsln` from it.
